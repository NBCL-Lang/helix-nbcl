# Helix NBCL Language Support

Tree-sitter integration and syntax highlighting for [NBCL](https://ncbl-lang.github.io) in the Helix editor.

## Installation

Run the `install.py` with python and follow the steps.

```bash
$ python3 install.py
```

## Bugs

Please report the bugs to [tree-sitter-nbcl](https://github.com/nbcl-lang/tree-sitter-nbcl) repository.
